# Process Warden

Process Warden is a premium Android utility for killing background apps with Shizuku or root access.

This repository is the official distribution channel for signed Process Warden APK releases. It does not contain source code.

## Licensing

- Process Warden is a paid product.
- On first launch, the app requires EULA acceptance and a valid Polar license key.
- Billing, trial handling, subscription management, and device deactivation are managed through Polar.
- After a successful validation, the app can continue working offline for up to 14 days before another live validation is required.

## Install and Update

- Download the latest signed APK from the Releases page.
- Enable installation from unknown sources on your Android device if prompted.
- Install the APK manually.
- In-app OTA updates are distributed through this repository's GitHub Releases feed.
